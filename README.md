# DRM_Watch
Atduino-based watch with Nokia 1280 LCD.

Used components:
- Arduino Nano (MEGA328P chip)
- Nokia 1280 LCD (PCF8814 chip)
- RTC module (DS3231M chip)
- Li-Po charger module (TC4056A chip)


To get components I've desoldered modules.
